# UI_INTERACTION_COPY_DRAFT

## 文档目的
统一全局搜索与关键安全流程的交互文案，确保提示清晰、可执行、可追溯。

## 当前状态
- 状态：v0.1 Draft（未冻结）
- 适用范围：Global Search、确认弹层、错误反馈、任务重试

## 文案原则（Draft）
1. 先结论，后原因，再给下一步。
2. 避免模糊词，优先动作化建议。
3. 高风险文案必须包含影响范围和回滚提示。

## 全局搜索文案（Draft）

### 空状态
- 标题：`开始搜索命令、软件或任务`
- 说明：`示例：视频剪辑工具、failed jobs、download start`

### 无结果
- 标题：`未找到匹配结果`
- 说明：`可尝试更短关键词或切换分组筛选`

### 加载中
- 标题：`正在检索...`

## 安全确认文案（Draft）

### 高风险动作确认
- 标题：`此操作需要确认`
- 内容：`该操作可能影响系统稳定性。请确认风险并继续。`
- 附加：`建议先执行 dry-run。`
- 按钮：`确认执行` / `取消`

### 门禁阻断
- 标题：`操作被安全策略阻断`
- 内容：`当前未满足执行条件：confirm / gate / approval record。`
- 下一步：`请先完成审批并开启 gate。`

## 下载相关文案（Draft）

### 来源阻断
- 标题：`下载来源未受信任`
- 内容：`当前域名不在白名单中，已阻断下载。`
- 下一步：`更换受信来源或提交审批。`

### 哈希校验失败
- 标题：`文件校验失败`
- 内容：`下载文件与期望哈希不一致，已阻断后续安装。`
- 下一步：`请重新下载并校验来源。`

## 任务重试文案（Draft）

### 可重试失败
- 标题：`任务执行失败`
- 内容：`你可以重试该任务，或查看错误详情。`
- 按钮：`立即重试` / `查看详情`

### 死信任务
- 标题：`任务已进入死信队列`
- 内容：`已达到最大重试次数，需要人工处理。`
- 下一步：`检查参数或环境后再手动重试。`

## AI 助手文案（Draft）

### 推荐结果
- 标题：`已生成推荐方案`
- 内容：`以下建议基于本地软件状态与风险规则。`

### 修复方案
- 标题：`已生成修复计划（仅计划）`
- 内容：`当前不会自动执行修复操作。`
- 下一步：`请审阅步骤后再执行。`

## 错误文案映射（Draft）
1. `validation`：`参数无效，请检查输入后重试。`
2. `security`：`操作被安全策略阻断。`
3. `integration`：`外部依赖异常，请稍后重试。`
4. `partial_success`：`部分任务已完成，请查看详情。`
